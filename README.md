## 🎯 დავალებების დონეები

| დონე | რაოდენობა | რა ისწავლი |
|------|-----------|-----------|
| 🟢 **მარტივი** (1–4) | 4 დავალება | `display: grid`, `grid-template-columns`, `gap`, `repeat()` |
| 🟡 **საშუალო** (5–9) | 5 დავალება | `grid-template-areas`, `auto-fit`, `minmax()`, `span`, media queries |
| 🔴 **რთული** (10–15) | 6 დავალება | Nested grids, `grid-auto-flow: dense`, 12-column system, რეალური აპლიკაციები |
