# CSS Grid - დავალებები სტუდენტებისთვის

დავალებები დაყოფილია სამ დონედ: **მარტივი**, **საშუალო**, **რთული**. თითოეული დავალება მოიცავს HTML სტრუქტურის შექმნას და CSS Grid-ის გამოყენებას მისი დასტილისთვის.

---

## 🟢 მარტივი დონე

### დავალება 1: სამი სვეტი
შექმენი `div` კონტეინერი, რომელშიც იქნება 3 ბავშვი ელემენტი. გამოიყენე CSS Grid და განათავსე ისინი 3 თანაბარი სვეტის სახით.

**მოთხოვნები:**
- გამოიყენე `display: grid`
- გამოიყენე `grid-template-columns: 1fr 1fr 1fr` ან `repeat(3, 1fr)`
- დაამატე ფონის ფერი თითოეულ ელემენტს
- ელემენტებს შორის დატოვე 10px მანძილი (`gap`)

---

### დავალება 2: ორი მწკრივი
შექმენი grid კონტეინერი, რომელშიც იქნება 6 ელემენტი დაყოფილი 2 მწკრივად და 3 სვეტად.

**მოთხოვნები:**
- გამოიყენე `grid-template-rows` და `grid-template-columns`
- თითოეული უჯრედის სიმაღლე იყოს 100px
- გამოიყენე `gap: 15px`

---

### დავალება 3: კალკულატორის ღილაკები
შექმენი მარტივი კალკულატორის ღილაკების განლაგება (0-9 ციფრები + ოპერაციები).

**მოთხოვნები:**
- 4 სვეტი
- ღილაკები იყოს კვადრატული ფორმის
- გამოიყენე `gap` ღილაკებს შორის

---

### დავალება 4: ფოტო გალერეა (მარტივი)
შექმენი 9 ფოტოს გალერეა 3x3 ბადეში.

**მოთხოვნები:**
- გამოიყენე `repeat(3, 1fr)`
- ფოტოები იყოს ერთნაირი ზომის
- დაამატე `border-radius` და `box-shadow`

---

## 🟡 საშუალო დონე

### დავალება 5: ბლოგის ლეიაუთი
შექმენი ბლოგის გვერდი შემდეგი სტრუქტურით: header, sidebar, main content, footer.

**მოთხოვნები:**
- გამოიყენე `grid-template-areas`
- Header და Footer გადაიჭიმოს მთელ სიგანეზე
- Sidebar იყოს მარცხნივ (250px)
- Main content აიღოს დარჩენილი სივრცე

```
+-------------------+
|     header        |
+-------+-----------+
| side  |   main    |
| bar   |  content  |
+-------+-----------+
|     footer        |
+-------------------+
```

---

### დავალება 6: პროდუქტების ბარათები
შექმენი 6 პროდუქტის ბარათი, რომელიც ავტომატურად რეგულირდება ეკრანის ზომის მიხედვით.

**მოთხოვნები:**
- გამოიყენე `grid-template-columns: repeat(auto-fit, minmax(250px, 1fr))`
- ყოველი ბარათი შეიცავდეს: სურათი, სათაური, ფასი, ღილაკი
- გამოიყენე `gap: 20px`

---

### დავალება 7: ჟურნალის სტილში განლაგება
შექმენი ჟურნალის სტილის გვერდი, სადაც ერთი ბარათი დიდია (გადაიჭიმოს 2 სვეტზე და 2 მწკრივზე), დანარჩენი კი - მცირე.

**მოთხოვნები:**
- 4 სვეტი, 3 მწკრივი
- პირველი ელემენტი: `grid-column: span 2; grid-row: span 2`
- სხვა ელემენტები: ჩვეულებრივი ზომის

---

### დავალება 8: დაშბორდი (Dashboard)
შექმენი ადმინისტრატორის დაშბორდი შემდეგი სექციებით:
- ზედა Navigation bar
- მარცხენა Sidebar (მენიუ)
- ცენტრალური სტატისტიკის ბარათები (4 ცალი)
- მარჯვენა აქტივობების panel

**მოთხოვნები:**
- გამოიყენე `grid-template-areas`
- სტატისტიკის ბარათები იყოს ცალკე grid კონტეინერი
- გამოიყენე ჩასმული (nested) grid

---

### დავალება 9: გასახდელი (Responsive) ლეიაუთი
შექმენი ლეიაუთი რომელიც იცვლება ეკრანის ზომის მიხედვით:
- **დიდი ეკრანი (>1024px):** 4 სვეტი
- **საშუალო ეკრანი (768px-1024px):** 2 სვეტი
- **მცირე ეკრანი (<768px):** 1 სვეტი

**მოთხოვნები:**
- გამოიყენე `@media` queries
- გამოიყენე `grid-template-columns`

---

## 🔴 რთული დონე

### დავალება 10: სრული ვებგვერდი - პორტფოლიო
შექმენი პორტფოლიოს ერთგვერდიანი საიტი, რომელიც შეიცავს:
1. **Hero სექცია** - 2 სვეტი (ტექსტი + სურათი)
2. **About** - 3 სვეტი (უნარები)
3. **Projects** - გალერეა `auto-fit` და `minmax`-ით
4. **Skills** - 6 ბარათი ანიმაციით
5. **Contact** - ფორმა და კონტაქტი (2 სვეტი)
6. **Footer** - 4 სვეტი

**მოთხოვნები:**
- სრულად Grid-ით აგებული
- Responsive ყველა ეკრანზე
- გამოიყენე `grid-template-areas` და `auto-fit`

---

### დავალება 11: მუსიკალური აპლიკაცია (Spotify-ის მსგავსი)
შექმენი მუსიკალური სტრიმინგ აპლიკაციის ინტერფეისი.

სტრუქტურა:
```
+---------+----------------+
| sidebar |   main view    |
|         |                |
|         |                |
+---------+----------------+
|       player bar         |
+--------------------------+
```

**მოთხოვნები:**
- მარცხენა Sidebar (250px) - მენიუ და პლეილისთები
- ცენტრალური ნაწილი - ალბომების grid (`auto-fill, minmax(180px, 1fr)`)
- ქვედა Player bar - 3 სვეტი (ტრეკის ინფო, კონტროლი, ხმის რეგულატორი)
- გამოიყენე `grid-template-areas`
- შავი თემა

---

### დავალება 12: მასონერული (Masonry) განლაგება
შექმენი Pinterest-ის სტილის გალერეა, სადაც ელემენტებს განსხვავებული სიმაღლე აქვთ.

**მოთხოვნები:**
- გამოიყენე `grid-template-rows: masonry` (ან ემულირება `grid-auto-rows`-ით)
- მინიმუმ 12 ფოტო/ბარათი სხვადასხვა სიმაღლით
- `grid-auto-flow: dense`
- Responsive დიზაინი

---

### დავალება 13: კალენდარი
შექმენი ერთი თვის კალენდარი CSS Grid-ის გამოყენებით.

**მოთხოვნები:**
- 7 სვეტი (კვირის დღეები)
- ზედა მწკრივი - დღეების სახელები (ორშ, სამ, ოთხ...)
- რიცხვები განათავსე სწორი დღის ქვეშ
- გამოიყენე `grid-column-start` თვის პირველი დღისთვის
- დღევანდელი დღე გამოყავი ფერით
- დაამატე "ღონისძიების" ნიშანი ზოგიერთ დღეზე

---

### დავალება 14: Trello-ს მსგავსი Kanban დაფა
შექმენი დავალებების მართვის დაფა (Kanban board) სვეტებით:
- To Do
- In Progress
- Review
- Done

**მოთხოვნები:**
- ჰორიზონტალური სქროლი მცირე ეკრანებზე
- თითოეული სვეტი - ცალკე grid კონტეინერი ბარათებისთვის
- ბარათები შეიცავდეს: სათაური, აღწერა, თეგები, მომხმარებლის ავატარი
- გამოიყენე nested grid

### დავალება 15: კომპლექსური Magazine Layout
შექმენი ციფრული ჟურნალის გვერდი ფიქსირებული, ასიმეტრიული ლეიაუთით.

**მოთხოვნები:**
- 12-სვეტიანი grid სისტემა
- მინიმუმ 8 სხვადასხვა ზომის სტატია
- ერთი მთავარი სტატია (გადაიჭიმოს 8 სვეტზე)
- ციტატის ბლოკი (გადაიჭიმოს 4 სვეტზე და 2 მწკრივზე)
- სხვადასხვა ფონის ფერი
- ტიპოგრაფიული იერარქია
- Responsive ქცევა

## 📚 დამატებითი რჩევები

### სასარგებლო CSS Grid თვისებები:
- `display: grid`
- `grid-template-columns` / `grid-template-rows`
- `grid-template-areas`
- `grid-column` / `grid-row` (span-ით)
- `gap` (ან `row-gap` / `column-gap`)
- `justify-items` / `align-items`
- `justify-content` / `align-content`
- `grid-auto-flow: dense`
- `repeat()`, `minmax()`, `auto-fit`, `auto-fill`
- `fr` ერთეული

### სასარგებლო რესურსები:
- [CSS Grid Garden](https://cssgridgarden.com/) - თამაშით სწავლა
- [MDN CSS Grid](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Grid_Layout)
- [Grid by Example](https://gridbyexample.com/)
